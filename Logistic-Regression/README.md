# Logistic Regression

## 📌 Overview

This notebook demonstrates the complete implementation of **Logistic Regression** using the **Breast Cancer Wisconsin Diagnostic Dataset**. The objective is to understand the end-to-end machine learning workflow for a binary classification problem.

---

## 🎯 Objective

- Understand Logistic Regression from theory to implementation.
- Build a binary classification model.
- Learn model evaluation using different classification metrics.

---

## 📂 Dataset

**Dataset:** Breast Cancer Wisconsin (Diagnostic)

**Problem Type:** Binary Classification

**Target Variable**

- 0 → Benign (No Cancer)
- 1 → Malignant (Cancer)
