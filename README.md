## 📖 Overview

This repository contains my **Machine Learning learning journey**, including theoretical concepts, practical implementations, and mini projects.

The objective is to build a strong understanding of Machine Learning algorithms from the ground up while applying them to real-world datasets.

The repository is continuously updated as I learn new algorithms and complete additional projects.

- Implement models using Scikit-Learn

# 💻 Installation

Clone the repository

```bash
git clone https://github.com/Rimsha35/Machine-Learning.git
```

Move into the project

```bash
cd Machine-Learning
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook

```bash
jupyter notebook
```
